組み込みシステム　「画像ビューアー」
【製作開始日】2013/11/16
【最終更新日】2013/11/18
【製　作　者】澤田 賢祐
【使　い　方】

・ハードウェア回路
	既存の回路構成における(emb01)からの変更点は以下の一点のみである。

	・ソースコードファイル /emb01/rtl/minilcd.v

	program.zip解等後、/rtl以下にminilcd.vが置かれているので、本ファイルを元の同名ファイルと置き換え、
	通常通り論理合成を行い、生成されたmieruemb.bitファイルをMieruEmbシステム上のPROMに書き込む。
	念のため、手元で論理合成したmieruemb.bitを同梱しておく。

・ソフトウェアプログラム
	program.zipを展開後、/app以下にてmakeを実行しコンパイル。
	生成されたinit.binをフォーマット済みのMMCに書き込み。
	その後、任意の枚数の縦横サイズがそれぞれ128pxの24bitカラービットマップ画像をMMCに書き込み、
	通常通りプログラムを起動する。起動後先頭の画像が表示される。スイッチ1で次の画像を参照できる。
	MMC内の全ての画像を一巡すると最初の画像に戻る。
	念のため、手元でコンパイルしたinit.binを同梱しておく。
